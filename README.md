# Digital Attendance System

Course: EEE227  
Programme: HND Electrical Engineering (Level 200)

This is a console-based C++ application for managing attendance in hour-based lectures.

Features:
- Student registration
- Student search
- Lecture session creation
- Attendance marking (Present, Absent, Late)
- Attendance summary reports
- Persistent file storage using fstream

How to Run:
1. Compile using g++
2. Run the executable
3. Follow menu instructions
